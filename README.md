# GoTransit - Bus Booking Website

Bus company website with booking system. Built with Next.js and Tailwind.

## Features

- Responsive design works on mobile
- Animated UI elements
- Book tickets online
- Select seats visually
- Multiple payment options
- Routes between Djibouti and Ethiopia

## Routes

Main routes:
- Djibouti City to Addis Ababa
- Addis Ababa to Dire Dawa
- Various local city routes

## Pages

- Home page with services
- Booking form
- Search results for buses
- Seat selection
- Payment checkout
- About company page

## Setup

Install packages:
```bash
npm install
```

Run dev server:
```bash
npm run dev
```

Go to http://localhost:3000

## Tech

- Next.js 14
- React
- TypeScript  
- Tailwind
- Framer Motion
